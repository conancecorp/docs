# Vue d'ensemble des fonctionnalités

Conance offre un ensemble complet de fonctionnalités pour gérer vos commissions et vos données métier.

## Fonctionnalités principales

### Import de données

Le cœur de Conance : importez vos relevés de commissions depuis vos partenaires.

- Support des fichiers **CSV** et **Excel**
- Mapping intelligent des colonnes
- Configurations réutilisables
- Correspondance intelligente des données

→ [En savoir plus sur l'import](/features/import/)

### Tableau de bord

Visualisez vos performances en un coup d'œil.

- Statistiques clés (commissions, contrats, clients)
- Graphiques interactifs
- Widgets personnalisables

→ [En savoir plus sur le tableau de bord](/features/dashboard)

### Gestion des commissions

Gérez l'ensemble de vos commissions avec des outils puissants.

- Filtrage avancé multi-critères
- Modification et suppression en masse
- Export de données

→ [En savoir plus sur les commissions](/features/commissions)
